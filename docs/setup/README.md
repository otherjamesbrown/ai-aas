# Setup Documentation

This directory contains documentation for setting up a new development environment for the AI-AAS platform.

## Documentation Files

- **[SETUP_CHECKLIST.md](./SETUP_CHECKLIST.md)** - Comprehensive checklist of all components, tools, keys, and configuration needed for local development, build, and testing.

- **[INSTALL_REQUIREMENTS.md](./INSTALL_REQUIREMENTS.md)** - Specific installation instructions for Linux (Ubuntu/Debian) with commands and scripts for installing required tools.

- **[MISSING_CREDENTIALS.md](./MISSING_CREDENTIALS.md)** - Complete guide on all required and optional API keys, tokens, and credentials (GitHub, Linode, observability, etc.).

- **[QUICK_AUTH_SETUP.md](./QUICK_AUTH_SETUP.md)** - Quick reference for setting up authentication (SSH keys, GitHub CLI, Linode tokens).

- **[SETUP_STATUS.md](./SETUP_STATUS.md)** - Current progress tracking document (useful during active setup).

## Quick Start

1. **Review the checklist**: Start with [SETUP_CHECKLIST.md](./SETUP_CHECKLIST.md) to understand all requirements.
2. **Install tools**: Follow [INSTALL_REQUIREMENTS.md](./INSTALL_REQUIREMENTS.md) for installation commands.
3. **Set up authentication**: Use [QUICK_AUTH_SETUP.md](./QUICK_AUTH_SETUP.md) for credentials setup.
4. **Verify missing credentials**: Check [MISSING_CREDENTIALS.md](./MISSING_CREDENTIALS.md) for any remaining tokens/keys.
