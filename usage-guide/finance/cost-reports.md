# Cost Reports

## Overview

This guide covers generating cost reports, analyzing cost data, and identifying cost optimization opportunities.

## Cost Report Types

### Summary Reports

- High-level cost overview
- Key cost metrics
- Cost trends
- Cost breakdown

### Detailed Reports

- Granular cost data
- Line-item costs
- Cost by service/model
- Cost by time period

## Generating Cost Reports

### Report Generation

1. Navigate to Reports section
2. Click "Cost Reports"
3. Select report type
4. Configure filters
5. Set time range
6. Generate report

### Report Configuration

- **Time Range**: Select period
- **Filters**: Service, model, project
- **Grouping**: Group by dimension
- **Metrics**: Select cost metrics
