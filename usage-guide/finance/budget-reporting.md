# Budget Reporting

## Overview

This guide covers generating budget reports, analyzing budget performance, and reporting to stakeholders.

## Budget Reports

### Report Types

- **Budget Summary**: High-level budget overview
- **Detailed Budget**: Detailed budget breakdown
- **Variance Report**: Budget vs. actual spending
- **Forecast Report**: Projected budget usage

### Generating Reports

1. Navigate to Budgets section
2. Click "Generate Report"
3. Select report type
4. Configure filters
5. Set time range
6. Generate report

## Report Analysis

### Budget Performance

- Compare budget to actuals
- Analyze variances
- Identify trends
- Forecast future spending

### Variance Analysis

- Identify significant variances
- Analyze causes
- Take corrective action
- Document findings

## Report Distribution

### Report Delivery

- **Email**: Send via email
- **Portal**: Available in portal
- **Export**: Download manually
- **Scheduled**: Regular delivery

### Report Recipients

- Finance team
- Management
- Department heads
- Stakeholders

## Best Practices

### Report Generation

- Generate reports regularly
- Use appropriate report types
- Include relevant metrics
- Provide context

### Report Analysis

- Review reports promptly
- Analyze variances
- Identify trends
- Take action on findings
