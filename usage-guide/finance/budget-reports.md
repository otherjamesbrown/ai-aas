# Budget Reports

## Overview

This guide covers generating budget reports, analyzing budget performance, and reporting budget status.

## Budget Report Types

### Budget Summary

- High-level budget overview
- Budget vs. actuals
- Budget remaining
- Budget status

### Detailed Budget Report

- Line-item budget breakdown
- Budget by category
- Budget by period
- Variance analysis

## Generating Budget Reports

### Report Generation

1. Navigate to Budgets section
2. Click "Generate Report"
3. Select report type
4. Configure filters
5. Set time range
6. Generate report

### Report Configuration

- **Budget**: Select budget
- **Time Range**: Select period
- **Filters**: Apply filters
- **Metrics**: Select metrics

## Budget Analysis

### Budget Performance

- Compare budget to actuals
- Analyze variances
- Identify trends
- Forecast future spending

### Variance Analysis

- Identify significant variances
- Analyze causes
- Take corrective action
- Document findings

## Report Distribution

### Report Delivery

- **Email**: Send via email
- **Portal**: Available in portal
- **Export**: Download manually
- **Scheduled**: Regular delivery

### Report Recipients

- Finance team
- Management
- Budget owners
- Stakeholders

## Best Practices

### Budget Reporting

- Generate reports regularly
- Analyze variances
- Identify trends
- Take action on findings

### Budget Management

- Monitor budgets closely
- Prevent overages
- Adjust as needed
- Document changes
