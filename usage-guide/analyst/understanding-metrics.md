# Understanding Metrics

## Overview

This guide covers understanding available metrics, their definitions, and how to interpret them.

## Metric Categories

### Usage Metrics

- **Total Requests**: Number of API requests
- **Successful Requests**: Successful API calls
- **Failed Requests**: Failed API calls
- **Token Usage**: Tokens consumed

### Cost Metrics

- **Total Cost**: Total spending
- **Cost per Request**: Average cost per request
- **Cost by Model**: Cost breakdown by model
- **Cost Trends**: Cost over time

### Performance Metrics

- **Response Time**: Average response times
- **Success Rate**: Percentage of successful requests
- **Error Rate**: Percentage of failed requests
- **Throughput**: Requests per time period

## Metric Definitions

### Request Metrics

- **Total Requests**: Total number of API requests made
- **Successful Requests**: Requests that completed successfully
- **Failed Requests**: Requests that failed or errored
- **Request Rate**: Requests per second/minute/hour

### Token Metrics

- **Input Tokens**: Tokens in request prompts
- **Output Tokens**: Tokens in responses
- **Total Tokens**: Sum of input and output tokens
- **Tokens per Request**: Average tokens per request

### Cost Metrics

- **Total Cost**: Total spending in currency
- **Cost per Token**: Cost per token consumed
- **Cost per Request**: Average cost per request
- **Cost by Model**: Cost breakdown by AI model

## Interpreting Metrics

### Trend Analysis

- Analyze trends over time
- Identify patterns
- Compare periods
- Forecast future metrics

### Comparative Analysis

- Compare metrics across dimensions
- Benchmark against targets
- Identify outliers
- Analyze variances

## Best Practices

### Metric Analysis

- Understand metric definitions
- Use appropriate time ranges
- Compare relevant periods
- Consider context

### Interpretation

- Look for patterns
- Identify anomalies
- Consider multiple metrics
- Validate findings
