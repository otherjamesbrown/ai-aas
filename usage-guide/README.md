# Usage Guide

Welcome to the AI-as-a-Service (AIaaS) Platform Usage Guide. This documentation is organized by persona to help you quickly find the information relevant to your role.

## Overview

The AIaaS platform provides inference-as-a-service capabilities, enabling organizations to manage AI model access, users, budgets, API keys, and usage analytics through a secure, multi-tenant platform.

## Documentation Structure

This guide is organized by persona, with each persona having:
- A main guide document covering role overview and common workflows
- Sub-documents for specific tasks and detailed procedures
