# Glossary

## Overview

This glossary defines key terms and concepts used throughout the AIaaS platform documentation.

## A

### API Key
A credential used to authenticate API requests to the platform. API keys are scoped to organizations and can have specific permissions.

### API Router Service
The service that routes inference requests to appropriate model backends, handles authentication, and enforces quotas.

### Analytics Service
The service that processes usage events, generates analytics, and provides reporting capabilities.

## B

### Budget
A spending limit configured for an organization to control costs and prevent overage.

### Budget Policy
Configuration that defines budget limits, alert thresholds, and enforcement actions.

## D

### Declarative Configuration
Git-as-source-of-truth configuration management where system state is defined in version control and automatically reconciled.

### Drift Detection
The process of detecting differences between declared configuration in Git and actual system state.

## I

### Inference
The process of generating AI model responses to user prompts via API requests.

## M

### MFA (Multi-Factor Authentication)
An authentication method requiring multiple verification factors for enhanced security.

### Model Registry
A catalog of available AI models and their revisions that can be used for inference.

## O

### Organization
A tenant-level entity that groups users, API keys, budgets, and usage data.

### Organization Admin
A user role with administrative privileges within an organization.

## R

### Role
A set of permissions that define what actions a user can perform within the platform.

### Rate Limit
A restriction on the number of API requests that can be made within a time period.

## T

### Token
A unit of text processing used by AI models. Tokens are consumed from budgets and incur costs.

## U

### Usage Event
A record of platform usage, including requests, tokens consumed, and costs.

### User-Org Service
The service that manages organizations, users, roles, and API keys.

## W

### Web Portal
The browser-based interface for managing organizations, members, budgets, API keys, and viewing usage insights.
