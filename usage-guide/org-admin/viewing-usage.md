# Viewing Usage

## Overview

This guide covers accessing and understanding usage analytics and reports in the web portal.

## Usage Dashboard

### Dashboard Overview

- **Current Usage**: Real-time usage metrics
- **Usage Trends**: Historical usage patterns
- **Cost Overview**: Spending summary
- **Key Metrics**: Important KPIs

### Dashboard Filters

- **Time Range**: Last 24 hours, 7 days, 30 days, custom
- **Service**: Filter by service
- **Model**: Filter by AI model
- **Project**: Filter by project (if configured)

## Usage Metrics

### Request Metrics

- **Total Requests**: Number of API requests
- **Successful Requests**: Successful API calls
- **Failed Requests**: Failed API calls
- **Request Rate**: Requests per second

### Token Metrics

- **Tokens Used**: Total tokens consumed
- **Tokens per Request**: Average tokens
- **Input Tokens**: Tokens in requests
- **Output Tokens**: Tokens in responses

### Cost Metrics

- **Total Cost**: Total spending
- **Cost per Request**: Average cost
- **Cost by Model**: Breakdown by model
- **Cost Trends**: Cost over time

## Usage Reports

### Generating Reports

1. Navigate to Usage section
2. Select report type
3. Configure filters
4. Set time range
5. Generate report
6. Export or view

### Report Types

- **Summary Report**: High-level overview
- **Detailed Report**: Detailed breakdown
- **Cost Report**: Cost analysis
- **Trend Report**: Usage trends

### Exporting Reports

- Export to CSV
- Export to PDF
- Export to Excel
- Schedule regular exports
