# Inviting and Managing Members

## Overview

This guide covers inviting new members to your organization, managing member accounts, and handling member lifecycle.

## Inviting Members

### Invitation Process

1. Navigate to Members section in web portal
2. Click "Invite Member"
3. Enter member email address
4. Select role for the member
5. Send invitation

### Invitation Email

- Member receives invitation email
- Invitation includes acceptance link
- Invitation expires after 7 days (configurable)
- Can resend invitation if needed

### Member Roles

- **Owner**: Full organization control
- **Admin**: Administrative access
- **Manager**: Team management access
- **Analyst**: Read-only analytics access
- **Developer**: API access

## Managing Members

### Viewing Members

- List all organization members
- View member status (active, invited, suspended)
- See last login time
- View assigned roles

### Updating Member Information

- Update member email (requires verification)
- Change member display name
- Update member roles
- Modify member permissions

### Suspending Members

- Temporarily disable member access
- Member cannot sign in
- Can be reactivated later
- Audit log entry created

### Removing Members

- Permanently remove member from organization
- Requires confirmation
- Member loses all access immediately
- Audit log entry created

## Member Status

### Active

- Member has accepted invitation
- Can sign in and access organization
- Full access based on role

### Invited

- Invitation sent but not accepted
- Can resend invitation
- Invitation expires after configured period

### Suspended

- Member access temporarily disabled
- Can be reactivated by admin
- Audit log shows suspension reason

## Best Practices

### Member Management

- Regularly review member list
- Remove inactive members
- Update roles as needed
- Monitor member activity

### Security

- Use least privilege principle
- Regularly review member access
- Monitor for suspicious activity
- Require MFA for sensitive roles
