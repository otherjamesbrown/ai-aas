# Setting Budgets

## Overview

This guide covers configuring budgets and spending limits for your organization to control costs.

## Budget Overview

### What Are Budgets?

- Spending limits for your organization
- Controls platform usage costs
- Prevents unexpected charges
- Configurable per organization

### Budget Types

- **Token Budget**: Limits token consumption
- **Request Budget**: Limits API requests
- **Cost Budget**: Limits spending in currency
- **Time-Based Budget**: Limits spending per time period

## Setting Up Budgets

### Via Web Portal

1. Navigate to Budgets section
2. Click "Create Budget"
3. Select budget type
4. Set budget amount
5. Configure alert thresholds
6. Set enforcement action
7. Save budget

### Budget Configuration

- **Budget Name**: Descriptive name
- **Budget Amount**: Spending limit
- **Alert Thresholds**: Warning levels (e.g., 50%, 80%, 100%)
- **Enforcement Action**: What happens at limit (warn, block, notify)
- **Time Period**: Daily, monthly, yearly

## Budget Enforcement

### Warning Mode

- Sends alerts at thresholds
- Allows continued usage
- Provides visibility
- No blocking

### Blocking Mode

- Blocks requests at limit
- Prevents overage
- Requires budget increase
- Stops all usage

### Notification Mode

- Sends notifications
- Tracks usage
- Reports to administrators
- No enforcement
