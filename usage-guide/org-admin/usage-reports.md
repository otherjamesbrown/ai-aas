# Usage Reports

## Overview

This guide covers generating, customizing, and exporting usage reports.

## Report Types

### Summary Reports

- High-level usage overview
- Key metrics and KPIs
- Cost summary
- Usage trends

### Detailed Reports

- Granular usage data
- Request-level details
- Token-level breakdown
- Cost breakdown

### Custom Reports

- Configure custom metrics
- Select specific data points
- Customize time ranges
- Add filters and groupings

## Generating Reports

### Via Web Portal

1. Navigate to Usage section
2. Click "Generate Report"
3. Select report type
4. Configure filters
5. Set time range
6. Generate report

### Report Configuration

- **Report Name**: Descriptive name
- **Time Range**: Select period
- **Filters**: Apply filters
- **Grouping**: Group by dimension
- **Metrics**: Select metrics

## Report Scheduling

### Scheduled Reports

- Schedule regular reports
- Set delivery frequency
- Configure recipients
- Choose delivery format

### Report Delivery

- **Email**: Send via email
- **Portal**: Available in portal
- **Webhook**: Send to webhook
- **Export**: Download manually

## Report Formats

### Available Formats

- **CSV**: Comma-separated values
- **PDF**: Portable document format
- **Excel**: Microsoft Excel format
- **JSON**: JSON format (API)

### Format Selection

- Choose format based on use case
- CSV for data analysis
- PDF for sharing
- Excel for detailed analysis

## Report Customization

### Custom Metrics

- Select specific metrics
- Calculate derived metrics
- Add custom calculations
- Include comparisons

### Custom Filters

- Filter by service
- Filter by model
- Filter by project
- Filter by user/API key

## Exporting Reports

### Export Options

- Download immediately
- Email report
- Schedule export
- API access

### Export Limits

- Maximum rows per export
- Maximum time range
- Rate limits
- Data retention

## Best Practices

### Report Generation

- Use appropriate report type
- Set relevant time ranges
- Apply useful filters
- Schedule regular reports

### Report Analysis

- Review reports regularly
- Identify trends
- Compare periods
- Take action on insights
