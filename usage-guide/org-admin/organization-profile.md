# Organization Profile

## Overview

This guide covers managing your organization's profile information, including name, contact details, and settings.

## Organization Information

### Basic Information

- **Organization Name**: Display name for your organization
- **Slug**: URL-friendly identifier (immutable after creation)
- **Description**: Organization description
- **Website**: Organization website URL

### Contact Information

- **Billing Contact**: Name, email, phone
- **Technical Contact**: Name, email, phone
- **Support Contact**: Name, email, phone

### Address Information

- **Street Address**: Physical address
- **City**: City name
- **State/Region**: State or region
- **Postal Code**: ZIP or postal code
- **Country**: Country code

## Updating Organization Profile

### Via Web Portal

1. Navigate to Organization Settings
2. Click "Edit Profile"
3. Update information
4. Review changes
5. Save updates

### Profile Updates

- Most fields can be updated
- Slug cannot be changed
- Some changes require confirmation
- Changes are logged in audit trail

## Organization Status

### Status Types

- **Active**: Organization is active and operational
- **Suspended**: Temporarily disabled
- **Trial**: Trial period active
- **Closed**: Organization closed

### Status Management

- View current status
- Request status changes
- Understand status implications
- Contact support if needed

## Organization Settings

### Plan Tier

- **Starter**: Basic plan
- **Growth**: Growth plan
- **Enterprise**: Enterprise plan

### Feature Flags

- Enable/disable features
- Configure organization preferences
- Set policy flags
- Manage feature access

## Best Practices

### Profile Management

- Keep information up to date
- Use accurate contact information
- Update address for billing
- Review profile regularly

### Security

- Protect organization information
- Limit who can edit profile
- Review changes in audit log
- Verify contact information
