# Billing Configuration

## Overview

This guide covers managing billing information, payment methods, and invoice settings.

## Billing Information

### Billing Contact

- **Name**: Billing contact name
- **Email**: Billing email address
- **Phone**: Billing phone number
- **Address**: Billing address

### Payment Methods

- **Credit Card**: Add/update credit card
- **Bank Account**: Configure ACH (if available)
- **Invoice Terms**: Set payment terms
- **Billing Address**: Configure billing address

## Invoice Settings

### Invoice Preferences

- **Invoice Frequency**: Monthly, quarterly, annually
- **Invoice Format**: PDF, email, portal
- **Invoice Delivery**: Email, portal, mail
- **Tax Information**: Configure tax details

### Invoice Details

- **Invoice Numbering**: Custom numbering scheme
- **Payment Terms**: Net 30, Net 60, etc.
- **Currency**: Invoice currency
- **Tax Calculation**: Configure tax settings

## Payment Methods

### Adding Payment Methods

1. Navigate to Billing section
2. Click "Add Payment Method"
3. Enter payment details
4. Verify payment method
5. Set as default (optional)

### Updating Payment Methods

- Update credit card information
- Change billing address
- Update payment preferences
- Set default payment method
